🤖 AkiraBot

Bot WhatsApp multifunction berbasis Baileys dengan sistem plugin modular, panel, QRIS payment, pushkontak, product system, dan berbagai fitur lainnya.

---

✨ Features

- ⚡ Fast & Lightweight
- 🔌 Modular Plugin System
- 🛒 Product System
- 💳 QRIS Payment Auto Check
- 📡 PushKontak System
- 🧰 Tools Menu
- 🎮 Fun Features
- 📡 Panel Create System
- 🎨 Maker Features
- 🔥 Easy To Develop

---

📦 INSTALLATION TERMUX

1. Update Package

pkg update && pkg upgrade -y

---

2. Install Dependencies

pkg install git nodejs ffmpeg imagemagick yarn webp -y

---

3. Clone Repository

git clone https://github.com/ReyZ4YouXGod/AkiraBotNew.git

---

4. Masuk Folder Bot

cd AkiraBotNew

---

5. Install Module

npm install

Kalau error:

npm install --force

atau:

yarn install

---

6. Jalankan Bot

npm start

atau:

node index.js

---

⚠️ ERROR FIX

Module Tidak Ditemukan

Install manual:

npm install axios
npm install @whiskeysockets/baileys
npm install qrcode-terminal

---

Sticker Error

pkg install webp -y

---

🔥 AUTO RUN TERMUX (PM2)

Install PM2

npm install -g pm2

Jalankan Bot

pm2 start index.js

Cek Status

pm2 list

Save Session

pm2 save

---

🔄 UPDATE BOT

git pull
npm install
