🤖 AkiraBot

Bot WhatsApp multifunction berbasis Baileys dengan sistem plugin modular, panel, QRIS payment, pushkontak, product system, dan berbagai fitur lainnya.

---

✨ Features

- ⚡ Fast & Lightweight
- 🔌 Modular Plugin System
- 🛒 Product System
- 💳 QRIS Payment Auto Check
- 📡 PushKontak System
- 🧰 Tools Menu
- 🎮 Fun Features
- 📡 Panel Create System
- 🎨 Maker Features
- 🔥 Easy To Develop

---

📦 Installation

pkg update && pkg upgrade
pkg install git nodejs ffmpeg imagemagick -y

git clone https://github.com/ReyZ4YouXGod/AkiraBotNew.git
cd AkiraBot

npm install
npm start
